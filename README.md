# YT Screensaver Manager Releases

Public download and update metadata repository for YT Screensaver Manager.

This repository intentionally does not contain the application source code. It contains only public release assets and metadata used by the app update flow.

## Latest Download

- Latest metadata: https://raw.githubusercontent.com/zdwudm/YTScreensaverManager-Releases/main/latest.json
- Releases: https://github.com/zdwudm/YTScreensaverManager-Releases/releases

## Current Distribution Model

- The main development repository stays private.
- Public installers are published here as GitHub Releases.
- `latest.json` points the app to the newest available installer and changelog.
- The app should never contain a private GitHub token.

## Notes

Early builds may still trigger Microsoft SmartScreen warnings until the installer is code-signed and has reputation.
